# Gamma Frontend

Write like a doc present like a deck

## Technology Used
- NextJS
- Zustand
- GSAP for animations
- GSAP ScrollTrigger

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
